# R Portfolio

This portfolio showcases R projects focused on data cleaning, statistical analysis, visualization, regression, classification, customer churn analysis, credit risk modeling, forecasting, and dashboard ready business insights.

The goal of this portfolio is to demonstrate how R can be used to analyze business data, create clean reporting outputs, build statistical models, compare machine learning approaches, and communicate insights clearly.
